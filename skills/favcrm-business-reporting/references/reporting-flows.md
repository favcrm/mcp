# Reporting Flow Patterns

## Discovery

```bash
favcrm dashboard
favcrm tool describe get_dashboard_stats
favcrm tool describe list_bookings
favcrm tool describe get_invoice_stats
favcrm tool describe list_subscriptions
favcrm tool describe search_members
```

## Weekly Snapshot

Default date range: last 7 days in company timezone.

Include:

- revenue
- bookings by status
- new customers/members
- overdue invoices
- active and past-due subscriptions
- notable risks
- three recommended actions

## Risk Report

Look for:

- cancelled/no-show booking patterns
- overdue invoices
- past-due subscriptions
- inactive high-value customers
- failed or low-volume comms

State tool source and date range.
